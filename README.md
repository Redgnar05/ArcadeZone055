# 🕹️ Arcade Zone

Galería interactiva de juegos de Scratch con estética retro-arcade.  
Elige tu juego desde un carrusel visual y juega directamente en la página sin salir de ella.

---

## ✨ Características

- 🎯 Carrusel con efecto *peek* — la tarjeta activa se centra y se asoman las adyacentes  
- 🖼️ Portadas personalizables por juego  
- 🪟 Modal con iframe para jugar sin cambiar de página  
- 🔳 Botón de pantalla completa  
- 📱 Soporte para *swipe* en móvil  
- 🌌 Estética retro-arcade con *scanlines*, grid neón y tipografía pixel  
- ⚡ 100% Vanilla JS — sin frameworks ni dependencias
